# `Component`对象

## 概述

`Component`对象是Tabulator的核心功能，它允许你可以和表格的Components直接交互。

表格中的每一行、列、单元格和行分组都有一个`Component`对象。每个`component`提供了一些可调用的函数用来操作表格。

在Tabulator中`Components`在几乎所有的回调函数中是可用的。
